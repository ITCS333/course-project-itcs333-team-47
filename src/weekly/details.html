Week 1: Introduction to HTML

Starts on: 2024-01-15

Description & Notes

This week introduces the fundamental concepts of HTML (HyperText Markup Language), the backbone of all web pages. You will learn about the basic structure of HTML documents, common tags like headings, paragraphs, and links, as well as semantic HTML elements that improve accessibility and SEO. By the end of this week, you should be comfortable creating simple web pages with proper structure and meaningful content.
